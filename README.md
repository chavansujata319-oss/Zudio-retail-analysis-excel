# Zudio-retail-analysis-excel

This project analyzes Zudio retail store data using Microsoft Excel.

Objectives:
• Analyze store-level performance
• Understand manager impact on sales
• Identify top product categories
• Track monthly sales trends

Tools Used:
• Microsoft Excel
• Pivot Tables
• Charts
• Basic formulas

Key Insights:
• Identified high revenue store locations
• Found best performing product categories
• Analyzed customer behaviour patterns

This project demonstrates Excel-based data analysis and business insights.
